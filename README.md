# Loois iOS SDK

此仓库是 [Loois团队](https://github.com/LOOIS-IO/) 提供的方便开发者接入钱包的SDK. 

Loois iOS SDK提供了常规的以太坊钱包功能和基于[路印协议](https://loopring.org/)Relay的交易等功能及便捷API.
